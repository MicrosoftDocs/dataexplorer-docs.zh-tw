# <a name="azure-data-explorer-java-sdk"></a>Azure 資料總管 JAVA SDK

*Kusto JAVA 用戶端*程式庫提供使用 JAVA 查詢 Azure 資料總管叢集的功能。
Azure 資料總管 JAVA SDK 可在[kusto-java](https://github.com/Azure/azure-kusto-java)中取得。
