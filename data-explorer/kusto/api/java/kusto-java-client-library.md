# <a name="azure-data-explorer-java-sdk"></a>Azure 資料總管 JAVA SDK

Azure 資料總管 JAVA 用戶端程式庫提供使用 JAVA 來查詢 Azure 資料總管叢集的功能。
Azure 資料總管 JAVA SDK 可在 [azure kusto](https://github.com/Azure/azure-kusto-java)java 中取得。
