# <a name="sync-kusto"></a>同步 Kusto

Sync Kusto 是一種工具，可讓使用者同步處理各種不同的 Kusto 架構實體，例如資料表架構和預存函數。 這是在本機檔案系統、Kusto 資料庫和 Azure Dev Ops 存放庫之間進行的同步處理。

您可以[在 GitHub 上](https://github.com/microsoft/synckusto)取得同步 Kusto。
