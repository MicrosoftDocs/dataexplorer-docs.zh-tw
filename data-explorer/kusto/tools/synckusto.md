# <a name="sync-kusto"></a>同步 Kusto

Sync Kusto 是一種工具,使用戶能夠在本地檔案系統、Kusto 資料庫和 Azure 開發人員行動儲存庫之間同步各種 Kusto 架構實體(如表架構和存儲的函數)。
