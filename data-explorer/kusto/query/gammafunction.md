# <a name="gamma"></a>gamma()

計算[伽瑪函式](https://en.wikipedia.org/wiki/Gamma_function)

**語法**

`gamma(`*X.*`)`

**引數**

* *x*: 伽瑪函式參數

**傳回**

* x 的伽瑪函數。
* 有關計算日誌伽馬函數,請參閱[loggamma()](loggammafunction.md)。
