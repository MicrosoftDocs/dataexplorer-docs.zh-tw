# <a name="gamma"></a>gamma()

計算[gamma](https://en.wikipedia.org/wiki/Gamma_function)函式

## <a name="syntax"></a>語法

`gamma(`*x*`)`

## <a name="arguments"></a>引數

* gamma 函數的*x*：參數

## <a name="returns"></a>傳回

* X 的 Gamma 函數。
* 如需計算記錄檔-gamma 函數，請參閱[loggamma （）](loggammafunction.md)。
