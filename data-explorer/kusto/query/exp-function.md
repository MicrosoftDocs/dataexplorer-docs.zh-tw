# <a name="exp"></a>exp()

x 的基-e 指數函數,該函數被提升到冪 x:e_x。  

**語法**

`exp(`*X.*`)`

**引數**

* *x*: 實數,指數值。

**傳回**

* x 的指數值。
* 有關自然(基-e)對數,請參閱[log()](log-function.md)。
* 有關基2和基-10對數的指數函數,請參閱[exp2()](exp2-function.md) [,exp10()](exp10-function.md)
