# <a name="exp"></a>exp()

X 的 base-e 指數函式，其 e 會向乘冪 x： e ^ x。  

## <a name="syntax"></a>語法

`exp(`*x*`)`

## <a name="arguments"></a>引數

* *x*：指數的實數值。

## <a name="returns"></a>傳回

* X 的指數值。
* 如需自然（底數為 e）的對數，請參閱[log （）](log-function.md)。
* 對於基底2和基底10對數的指數函式，請參閱[exp2 （）](exp2-function.md)、 [exp10 （）](exp10-function.md)
