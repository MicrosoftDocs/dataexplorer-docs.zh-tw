# <a name="pow"></a>pow()

傳回冪的結果

## <a name="syntax"></a>語法

`pow(`*基底* `,`*指數*`)`

## <a name="arguments"></a>引數

* *基底*：基底值。
* *指數*：指數值。

## <a name="returns"></a>傳回

傳回以乘冪指數： base ^ 指數的基底。
