# <a name="pow"></a>pow()

傳回提升到電源的結果

**語法**

`pow(`*基礎*`,`*指數*`)`

**引數**

* *基值*:基值。
* *指數*:指數值。

**傳回**

將基升到功率指數:基 = 指數。
