# <a name="abs"></a>abs()

計算輸入的絕對值。  

## <a name="syntax"></a>語法

`abs(`*X*`)`

## <a name="arguments"></a>引數

* *x*：整數或實數或 timespan 值。

## <a name="returns"></a>傳回

* X 的絕對值。
