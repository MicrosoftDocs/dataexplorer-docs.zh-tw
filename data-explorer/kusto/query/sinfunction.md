# <a name="sin"></a>sin()

傳回正弦函數。

## <a name="syntax"></a>語法

`sin(`*x*`)`

## <a name="arguments"></a>引數

* *x*：實數。

## <a name="returns"></a>傳回

*  的結果。`sin(x)`
