# <a name="cos"></a>cos()

傳回余弦函數。

## <a name="syntax"></a>語法

`cos(`*x*`)`

## <a name="arguments"></a>引數

* *x*：實數。

## <a name="returns"></a>傳回

*  的結果。`cos(x)`
