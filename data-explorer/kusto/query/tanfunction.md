# <a name="tan"></a>tan()

傳回正切函數。

## <a name="syntax"></a>語法

`tan(`*x*`)`

## <a name="arguments"></a>引數

* *x*：實數。

## <a name="returns"></a>傳回

*  的結果。`tan(x)`
