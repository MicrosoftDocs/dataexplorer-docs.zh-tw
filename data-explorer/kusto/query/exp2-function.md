# <a name="exp2"></a>exp2()

X 的基底2指數函式，其為2的乘冪 x： 2 ^ x。  

## <a name="syntax"></a>語法

`exp2(`*x*`)`

## <a name="arguments"></a>引數

* *x*：指數的實數值。

## <a name="returns"></a>傳回

* X 的指數值。
* 如需自然（以2為基底）的對數，請參閱[log2 （）](log2-function.md)。
* 若為以 e 為底數的指數函數和以10為基底的對數，請參閱[exp （）](exp-function.md)、 [exp10 （）](exp10-function.md)
