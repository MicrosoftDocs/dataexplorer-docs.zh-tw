# <a name="exp2"></a>exp2()

x 的基-2指數函數,該函數為 2,提升為功率 x:2+x。  

**語法**

`exp2(`*X.*`)`

**引數**

* *x*: 實數,指數值。

**傳回**

* x 的指數值。
* 有關自然(基-2)對數,請參閱[log2()](log2-function.md)。
* 有關基 e 和基-10 對數的指數函數,請參閱[exp()](exp-function.md) [,exp10()](exp10-function.md)
