# <a name="exp2"></a>exp2()

X 的以2為底數的指數函式，其為2到乘冪 x： 2 ^ x。  

## <a name="syntax"></a>語法

`exp2(`*X*`)`

## <a name="arguments"></a>引數

* *x*：指數的實數和值。

## <a name="returns"></a>傳回

* X 的指數值。
* 針對自然 (base-2) 對數，請參閱 [log2 ( # B3 ](log2-function.md)。
* 若為 base-e 和 base-10 對數的指數函式，請參閱 [exp ( # B1 ](exp-function.md)， [Exp10 ( # B3 ](exp10-function.md)
