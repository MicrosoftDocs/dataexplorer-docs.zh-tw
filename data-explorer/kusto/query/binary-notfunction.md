# <a name="binary_not"></a>binary_not()

傳回輸入值的位否定。

```kusto
binary_not(x)
```

## <a name="syntax"></a>語法

`binary_not(`*num1*`)`

## <a name="arguments"></a>引數

* *num1*：數值 

## <a name="returns"></a>傳回

傳回數位的邏輯 NOT 運算： num1。
