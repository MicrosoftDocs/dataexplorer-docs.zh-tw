# <a name="binary_not"></a>binary_not()

返回對輸入值的一點否定。

```kusto
binary_not(x)
```

**語法**

`binary_not(`*num1*`)`

**引數**

* *公尺元1*:數字 

**傳回**

返回對數位的邏輯 NOT 操作:num1。
