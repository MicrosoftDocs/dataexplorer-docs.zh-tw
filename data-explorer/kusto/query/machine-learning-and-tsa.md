# <a name="time-series-analysis"></a>時間序列分析 

Kusto 查詢語言以原生資料類型的形式提供序列支援。
Operator make 系列會將資料轉換成數列資料類型，並提供一系列函數來進行此資料類型的先進處理。 這些函式包括資料清理、機器學習模型和極端偵測。
