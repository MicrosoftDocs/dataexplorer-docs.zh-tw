# <a name="exp10"></a>exp10()

X 的以10為底數的指數函式，其為10到乘冪 x： 10 ^ x。  

## <a name="syntax"></a>語法

`exp10(`*X*`)`

## <a name="arguments"></a>引數

* *x*：指數的實數和值。

## <a name="returns"></a>傳回

* X 的指數值。
* 針對自然 (base-10) 對數，請參閱 [log10 ( # B3 ](log10-function.md)。
* 若為 base-e 和底數2對數的指數函式，請參閱 [exp ( # B1 ](exp-function.md)， [Exp2 ( # B3 ](exp2-function.md)
