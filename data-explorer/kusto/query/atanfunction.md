# <a name="atan"></a>atan()

傳回其正切為指定數位的角度（的反運算 [`tan()`](tanfunction.md) ）。

## <a name="syntax"></a>語法

`atan(`*x*`)`

## <a name="arguments"></a>引數

* *x*：實數。

## <a name="returns"></a>傳回

* 的反正切值`x`
