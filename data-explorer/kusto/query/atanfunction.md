# <a name="atan"></a>atan()

返回切線為指定數位([`tan()`](tanfunction.md)的反向操作)的角度。

**語法**

`atan(`*X.*`)`

**引數**

* *x*: 實數。

**傳回**

* 弧切線值`x`
