# <a name="cot"></a>cot()

計算指定角度的三角餘切（以弧度為單位）。

## <a name="syntax"></a>語法

`cot(`*X*`)`

## <a name="arguments"></a>引數

* *x*：實數。

## <a name="returns"></a>傳回

* 的餘切函數值 `x`
