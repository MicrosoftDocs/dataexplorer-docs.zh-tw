## <a name="query-data"></a>查詢資料

1. 選取 **v** 按鈕以開啟查詢。 複製到 Web UI 以編輯查詢。

1. 在右側功能表中，選取 [快速查詢]  和 [工具]  ： 

    * [快速查詢]  包含具有範例查詢的 Web UI 連結。
    * [工具]  包含 Web UI 上 **Drop 命令**的連結，其可讓您藉由執行相關的 `.drop` 命令，針對問題進行疑難排解。

    > [!TIP]
    > 您可能會在使用 `.drop` 命令時遺失資料。 請小心使用。
