## <a name="install-python-package"></a>安裝 Python 套件

若要為 Azure 資料總管 (Kusto) 安裝 Python 套件，請開啟在其路徑中有 Python 的命令提示字元。 執行以下命令：

```python
pip install azure-common
pip install azure-mgmt-kusto
```
